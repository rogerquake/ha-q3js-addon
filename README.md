# Q3JS Add-on Repository

Home Assistant add-on repository for **Q3JS - Quake III Arena**.

## Installation

1. Go to **Settings → Add-ons → Add-on Store → ⋮ → Repositories**
2. Add this URL: `https://github.com/rogerquake/ha-q3js-addon`
3. Find **Q3JS - Quake III Arena** and install it

## Game assets

Copy your `pak*.pk3` files to `/share/q3js/baseq3/` on your HA host before starting the add-on.

## Companion integration

Install the Lovelace card and HA entities via HACS:
`https://github.com/rogerquake/ha-q3js`
